# heading 태그 알아보기

## 개발자 도구를 head태그 지우기

- 개발자 도구 Console 박스 클릭
- `document.head.parentNode.removeChild(document.head);` 입력
- head 태그 사라짐 -> css도

## heading 태그

- **태그는 h1\~ h6** **중요도 순서 ( chrome 확장 프로그램 headingsmap )**
- **숫자가 높을수록 중요도 와 글씨 크기가 줄어듬**
